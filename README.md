Projeto Orientado a Objetos – Agregação em Python
Descrição

Este projeto tem como objetivo demonstrar o conceito de Agregação dentro da Programação Orientada a Objetos (POO) utilizando a linguagem Python.
O código foi desenvolvido com foco didático, visando facilitar o entendimento do relacionamento entre classes de forma clara e prática.



A agregação é um tipo de associação onde uma classe utiliza outra, mas ambas podem existir de forma independente.

Objetivo do Projeto

Praticar Programação Orientada a Objetos em Python

Compreender o conceito de agregação entre classes

Aplicar boas práticas de organização e relacionamento entre objetos

Desenvolver raciocínio arquitetural em POO

Conceito Trabalhado: Agregação

Na agregação:

Uma classe tem outra classe

O objeto agregado não depende do ciclo de vida do objeto principal

Os objetos podem existir separadamente

Exemplo conceitual:

Uma pessoa tem um celular

O celular pode existir sem a pessoa

A pessoa pode existir sem o celular
